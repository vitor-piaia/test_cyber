## üöÄ Project Configuration

### Prerequisites
- Docker
- Docker Compose

### Instalation

1. Open project directory:
   ```bash
   cd test_cyber
   ```

2. Up the project containers:
   ```bash
   docker-compose up -d --build
   ```

3. Copy the file example `.env`:
   ```bash
   cp .env.example .env
   ```

4. Access Laravel container:
   ```bash
   docker exec -it laravel bash
   ```

5. Install the PHP dependencies:
   ```bash
   composer install
   ```

6. Configure the `.env` file:
- Uncomment the database information and add:
  ```env
  DB_CONNECTION=pgsql
  DB_HOST=db_postgres
  DB_PORT=5432
  DB_DATABASE=laravel
  DB_USERNAME=sail
  DB_PASSWORD=password
  ```

7. Run the optimization and database commands:
   ```bash
   composer dump
   php artisan optimize
   php artisan migrate
   ```

8. Correct the permissions for the `storage/` folder inside the container:
   ```bash
   chown -R www-data:www-data storage/
   ```
